## Description

The Superheroes API is a Flask-based REST API used to track heroes and their superpowers.  
It allows users to view heroes, powers, assign powers to heroes, and update power descriptions.

This project demonstrates:

- Flask REST APIs
- SQLAlchemy models and relationships
- Validations
- Database migrations
- Pipenv environment management

---

## Technologies Used

- Python
- Flask
- Flask SQLAlchemy
- Flask Migrate
- SQLite
